# No_hoshi — Mint Tracker

Free mint coming soon. 888 supply. Forever onchain.

## Files
- `index.html` — Main page
- `bg.jfif` — Hero background art
- `kokoro1.jfif` — OG image (used for social previews)
- `vercel.json` — Vercel config

## Deploy to Vercel
1. Push this folder to GitHub
2. Go to vercel.com → Import repo → Deploy
3. No build settings needed — it's static HTML
